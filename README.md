# Bias Variance Tradeoff Atari Bot

This Python script demonstrates the bias-variance tradeoff concept in reinforcement learning using a simplified agent. It simulates an agent interacting with a gym environment and conceptually adjusts its 'bias' and 'variance' based on performance, illustrating how to balance underfitting and overfitting.

## Language

`python`

## How to Run

1. Install gymnasium: `pip install gymnasium`
2. Run the script: `python atari_bot.py`
